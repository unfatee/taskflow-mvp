# Настройка Bitrix24 для ПР-05

В этой версии проекта интеграция настроена под сущность Bitrix24 CRM **Сделка**.

## 1. Пользовательские поля сделки

Созданы 7 пользовательских полей сущности `CRM_DEAL`:

| Название поля | Тип | Код Bitrix24 |
|---|---|---|
| Описание задачи | string | `UF_CRM_1777804608701` |
| Дедлайн задачи | date | `UF_CRM_1777804782669` |
| Приоритет задачи | enumeration | `UF_CRM_1777804868815` |
| Категория задачи | string | `UF_CRM_1777804887989` |
| Статус задачи | enumeration | `UF_CRM_1777804927342` |
| Дата создания в приложении | date | `UF_CRM_1777804943652` |
| Источник заявки | string | `UF_CRM_1777804958149` |

Значения списка «Приоритет задачи»:

| Значение | ID |
|---|---|
| High | `44` |
| Medium | `46` |
| Low | `48` |

Значения списка «Статус задачи»:

| Значение | ID |
|---|---|
| new | `50` |
| in_progress | `52` |
| done | `54` |

## 2. Роли

Минимальный набор ролей для отчета:

- `Менеджер` — чтение, добавление и изменение своих сделок, без удаления и экспорта.
- `Администратор` — полный доступ к сделкам, настройкам, импорту, экспорту и роботам.

## 3. Автоматизация

Настроен робот на стадии `Новая`:

```text
Новая сделка -> Добавить уведомление -> Получатель: Ответственный
```

Текст уведомления:

```text
Создана новая сделка из веб-интерфейса TaskFlow. Проверьте данные задачи, дедлайн, приоритет и статус.
```

## 4. Входящий webhook

Откройте:

```text
Приложения -> Разработчикам -> Другое -> Входящий вебхук
```

Выберите право доступа:

```text
CRM
```

Скопируйте URL webhook. Он выглядит примерно так:

```text
https://your-domain.bitrix24.ru/rest/1/webhook_code/
```

Не вставляйте webhook в GitHub. Добавьте его в Vercel:

```text
BITRIX24_WEBHOOK_URL=https://your-domain.bitrix24.ru/rest/1/webhook_code/
CRM_MOCK_MODE=false
```

## 5. Проверка API вручную

После деплоя выполните тестовый запрос:

```bash
curl -X POST https://your-vercel-domain.vercel.app/api/crm/create-deal \
  -H "Content-Type: application/json" \
  -d '{
    "task": {
      "id": "test-001",
      "title": "Проверить API-интеграцию",
      "description": "Тестовая задача из ПР-05",
      "deadline": "2026-04-20",
      "priority": "high",
      "category": "Учеба",
      "status": "new",
      "createdAt": "2026-04-15T10:30:00.000Z",
      "source": "Web MVP / ПР-04"
    }
  }'
```

Успешный ответ:

```json
{
  "ok": true,
  "crm": "bitrix24",
  "entity": "deal",
  "dealId": 123,
  "mock": false,
  "message": "Задача успешно отправлена в Bitrix24 CRM. ID сделки: 123"
}
```

После успешной отправки в Bitrix24 появится новая сделка с названием:

```text
TaskFlow: Проверить API-интеграцию
```
